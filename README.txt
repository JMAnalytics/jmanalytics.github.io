# 🚲 Bike Sales Analysis – Excel Project

Welcome to my data analytics portfolio! This repository features my **Bike Sales Analysis** project, completed entirely in **Microsoft Excel**. It highlights my ability to clean, analyze, and visualize data using Excel tools like PivotTables, charts, and formulas.

---

## 📊 Project Overview

The goal of this project was to analyze bike sales data to uncover insights into overall sales trends. The final Excel dashboard provides a clear and interactive summary of key metrics for decision-makers.

---

## 🛠️ Tools & Skills Used

- **Excel PivotTables** – for dynamic summarization
- **Excel Charts** – for visual storytelling
- **Data Cleaning** – removing duplicates, fixing formatting
- **Conditional Formatting** – to highlight patterns
- **Formulas** – including `TRIM`IF`, and `LEN`.

---

## 🎯 Key Insights

- Most purchases were made by customers aged 31–45.
- Male customers accounted for the highest percentage of sales.
- Bike sales were higher among customers with higher income levels.
